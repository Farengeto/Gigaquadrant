GIGAQUADRANT

ABOUT<br>

CONTROLS<br>
''G'' - Display grid<br>
''H'' - Go to Capital<br>
''P'' - Pause game<br>
''X'' - Zoom out<br>
''Z'' - Zoom in<br>
''1-8'' - View faction's capital<br>
''<'' - Reduce game speed<br>
''>'' - Increase game speed<br>
''+'' - Add ship to queue<br>
''-'' - Remove ship from queue<br>
Arrow Keys - Move camera
