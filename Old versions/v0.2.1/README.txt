  CONTROLS
           G - Display grid
           H - Go to Capital
           P - Pause game
           X - Zoom out
           Z - Zoom in
         1-8 - View faction's capital
           < - Reduce game speed
           > - Increase game speed
           + - Add ship to quene
           - - Remove ship from quene
  Arrow Keys - Move camera
